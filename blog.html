Question-1: What are the different ways to select an element in the DOM?

Answer:The ways to select an element in the DOM are-
1. document.getElementsByTagName(); 2. document.getElementById(); 3. document.getElementsByClassName(); 4. document.querySelector(); 5. document.querySelectorAll();

Question-2: What are the difference between innerHTML, innerText, and innerContent?

Answer: The difference between innerHTML, innerText, and innerContent are-
1. innerHTML is working with HTML tags inside the element. 2. innerText is working only visible text, considering CSS styles. 3. innerContent is working for faster performance and when need all text, including hidden elements.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a pattern in JavaScript that allows you to handle events more efficiently by attaching a single event listener to a parent element instead of multiple child elements. This is possible due to event bubbling.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a procession in the DOM where an event starts from the target element and propagates up through its ancestors until it reaches the root document.
